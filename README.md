# Lark VPS — сервис аренды VPS с автовыдачей

## Технологии
- HTML/CSS/JS (адаптивный дизайн)
- Firebase Auth + Firestore
- ЮKassa API (СБП, карты)
- Nginx + PHP на VPS

## Что умеет
- Регистрация и вход (email + Google)
- Автоматическая выдача сервера после оплаты
- Чат поддержки
- Админ-панель
